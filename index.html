Bienvenido a mi página web.

Este sitio está siendo usado a modo de práctica para llevar a cabo las actividades propuestas en la asignatura Herramientas HTML y CSS I del Máster universitario de desarrollo de sitios y aplicaciones web de la Universitat Oberta de Catalunya.

En este primer ejercicio nos proponen realizar un sitio web (este) que contenga al menos dos categorias (Lenguajes FrontEnd y Lenguajes BackEnd) y que en cada categoría haya al menos una página de detalle.

Requisito también es que al menos una página tenga un mínimo de interactividad con JS, en nuestro caso lo hemos hecho en nuestro menú de navegación, el cual si cambias el tamaño del navegador verás como cambia de un menú de navegación en horizontal a un desplegable.

Publicaremos la web en internet, a través de GitHub y Netlify. Si estás leyendo esto es que, efectivamente, se ha conseguido publicar :) yey!.
