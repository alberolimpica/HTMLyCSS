Lenguajes FrontEnd

Cuando abrimos una página web en un navegador, cuando está carga lo primero que vemos es la parte FrontEnd del sitio.

El desarrollo FrontEnd se ocupa de diseñar la parte del lado del cliente, la que se verá desde el navegador.

Los lenguajes mas importantes FrontEnd son los siguientes:

  • HTML
  • CSS
  • JavaScript
