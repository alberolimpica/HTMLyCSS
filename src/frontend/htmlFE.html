HTML o HyperText Markup Language (lenguaje de marcas de hipertexto) es un lenguaje para el desarrollo web.

Mediante este lenguaje describimos la estructura de una pagina web utilizando una serie de elementos que le dicen al navegador como mostrar el contenido. Estos elementos se componen de etiquetas y, en ocasiones, atributos.

Las etiquetas llevan < y > y nos encontramos con una etiqueta al principio y al final del elemento, siendo que esta ultima la qie lleva </ y >. Entre las etiquetas de principio y de final encontramos nuestro contenido. Los atributos proveen información adicional a los elementos (id, clase, link...) y se especifican en la etiqueta principal.

Ejemplo de etiquetas: <p></p>

Ejemplo de etiquetas con atributo: <a href="../../index.html"></a>
